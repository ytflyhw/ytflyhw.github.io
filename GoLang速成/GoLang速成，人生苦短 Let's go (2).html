8、指针

& 即取地址， * 即根据地址取值

9、结构体

类似 JAVA 的类，结构体占用一块完整内存，其中字段首字母大写表示可以公开访问，小写表示私有

  • 实例化

也可以搞个构造函数

  • 嵌套结构体

一个结构体中可以嵌套包含另一个结构体或结构体指针

  • 方法和接收者

让结构体实现一些方法

注意：方法参数默认为值传递，如果需要修改参数中的字段则需要指明参数类型为指针

10、函数

函数也可以作为其他函数的参数或返回值

  • 变量作用域

全局 > 局部 > 代码块

用变量时，当前作用域找不到就往上找，当前作用域内的变量，外面访问不到

  • 闭包

闭包指的是一个函数和与其相关的引用环境组合而成的实体

f 接受了 adder() 返回的函数，并根据作用域向上找寻原则可以访问其外部作用域的x变量，在 f 的生命周期中， 变量 x 也一直有效， f 就是一个闭包。

  • defer 语句

类似 JAVA 的 fianly 语句，无论如何都会在其归属的函数即将返回时执行，且遵循先定义的后执行的原则（栈）。常用于关闭资源，或 recover() 异常。

11、接口 interface

一组方法的集合，只要一个结构体实现了其中的所有方法，即认为那个结构体实现了这个接口。

实现了接口的实体均可以被赋值给接口类型的变量

接口是一种类型

一个结构体可以实现多个接口

一个接口的方法，不一定需要一个结构体完全实现，可以尤其内嵌的其他结构体实现。

空接口不含任何方法，因此任何类型都实现了空接口（有些类似 JAVA Object类），作为参数时允许函数被传入任何类型的参数。

  • 断言

一个接口的值（简称接口值）是由一个具体类型和具体类型的值两部分组成的。这两部分分别称为接口的动态类型和动态值。

通过断言判断接口中值的类型
